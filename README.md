# gc-transmanage-frontend
this is for gc-transaction-management app front end

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### build css style

build tailwind css file from tailwind-styles.css

```sh
$ npm run build-css
```

### Serve
To start vite development server

```sh
$ npm run dev
```

### Build
Compile and build

```sh
$ npm run build
```
